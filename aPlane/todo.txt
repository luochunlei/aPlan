VER 0.1
--------
    首先从最简单的开始入手，整个界面就是显示一个列表，每个项目是任务。
    在最上方有一个工具栏，显示可以进行的功能：添加、删除。
    点击其中一个记录，扩大显示详细信息。点击其他的地方，回到列表状态。

= 1. 编写数据结构。
    + 1. 建立内部使用的数据结构。
    - 2. 保存到DB中。
    - 3. 从DB中读取。
    - 4. 编写业务处理接口，对外部。
    - 5. 和iCloud连动。
= 2. 加入中间的列表控件。
    + 1. 显示列表，列表中使用计划的数据。
    - 2. 显示某个计划的详细信息。
    + 3. 添加一个计划。
        - 1. 显示添加计划画面。
        - 2. 进行详细设定。
    - 4. 删除一个计划。
- 3. 考虑大量的计划时，按照天进行加载。
- 4. 设置画面。
- 5. 计划提醒功能。
